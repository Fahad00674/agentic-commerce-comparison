Agentic Commerce Landscape · May 2026

Agent Trust & Payment Frameworks — Comparison

Seven frameworks defining how AI agents are identified, trusted, and paid. Side-by-side: tech, fees, integration model, and — critically — what verifiable signal each emits that a trust-aggregator layer can ingest.

01 The Frameworks

BB
Asset Issuance New

Bitbond Token Tool MCP

bitbond.com · BaFin / Berlin

Open-source MCP server. AI agents (Claude, Cursor, Windsurf) deploy compliant ERC20/ERC1400 tokens via natural language. CertiK-audited.

OKX
Agent-to-Agent

OKX Agent Payments Protocol

okx.com / X Layer

Open standard for full agent commerce — quote, negotiate, escrow, settle, dispute. Stablecoin payments on X Layer (~5000 TPS, sub-cent fees).

VISA
Edge Trust

Visa TAP

developer.visa.com

Open standard so merchants can verify a real agent vs. a scraper.

MC
Card Rail

Mastercard Agent Pay

mastercard.com

Tokenized card payments for verified agents with KYA scoring.

x402
Crypto-Native New

x402 + GoPlausible

x402.org · goplausible.com

HTTP 402 "Payment Required" protocol. Stablecoin micropayments per request, Algorand-native facilitator available.

S
Sell-Side

Stripe Agentic Commerce

docs.stripe.com/agents

Stack to sell products inside AI agents — feed, checkout, payment.

SP
Dev Infra

Stripe Projects

docs.stripe.com/projects

CLI for agents to provision DBs, auth, analytics & pay providers.

03 Verun's Adoption Stance

How Verun positions against each framework

Verun is a signal-aggregation layer for regulated EU agent commerce — it consumes trust signals from multiple sources, computes a unified Accredited Agent Score (0–1000), and gates on-chain operations through validator consensus. Here's the stance on each framework:

Bitbond Token Tool MCP
BaFin-experienced, Berlin-based, ERC1400 native — direct fit for tokenforge gate. Agents can mint compliant securities; Verun gates the deploy via score.
Integrate
OKX Agent Payments Protocol
Agent-to-agent settlement, escrow, and dispute history → strong reputation signal; X Layer can co-exist with Algorand for non-securities flows.
Ingest
Visa TAP
Cryptographic-signature signal feeds directly into the Verun score as a verifiable identity input.
Ingest
MC Agent Pay (KYA)
4-tier KYA classification maps cleanly to Verun's tier model (LOW / MED / HIGH / BLOCK).
Ingest
x402 + GoPlausible
Algorand-native payment rail — wallet history + on-chain reputation as inputs; payment gate on accredited operations.
Ingest + Settle
Stripe Agentic Commerce
Built for retail D2C, not regulated securities. Useful only if Verun ever opens a fiat off-ramp surface.
Watch
Stripe Projects
Internal dev infra only. Not part of the agent-trust flow.
