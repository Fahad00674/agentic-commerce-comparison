# Agentic Commerce — Comparison

Static site comparing 5 agentic commerce frameworks:

- **Trusted Agents** (consultancy)
- **Visa Trusted Agent Protocol** (open spec)
- **Stripe Agentic Commerce Suite** (sell on agents)
- **Stripe Projects** (CLI for agent-driven infra)
- **Mastercard Agent Pay** (network tokens)

## Pages

- `index.html` — 60-second overview
- `comparison.html` — full feature matrix, fees, use cases

## Local preview

```bash
npx serve .
```

## Deploy on Vercel

```bash
npm i -g vercel
vercel --prod
```
